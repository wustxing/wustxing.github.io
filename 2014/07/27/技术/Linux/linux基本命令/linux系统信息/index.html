linux系统信息

2014-07-27
基本命令
  1. uname -a # 查看内核/操作系统/CPU信息
  2. head -n 1 /etc/issue # 查看操作系统版本
  3. cat /proc/cpuinfo # 查看CPU信息
  4. hostname # 查看计算机名
  5. lspci -tv # 列出所有PCI设备
  6. lsusb -tv # 列出所有USB设备
  7. lsmod # 列出加载的内核模块
  8. env # 查看环境变量
  9. arch 显示机器的处理器架构(1)
  10. uname -m 显示机器的处理器架构(2)
  11. uname -r 显示正在使用的内核版本
  12. dmidecode -q 显示硬件系统部件
  13. hdparm -i /dev/hda 罗列一个磁盘的架构特性
  14. hdparm -tT /dev/sda 在磁盘上执行测试性读取操作
  15. cat /proc/interrupts 显示中断
  16. cat /proc/meminfo 校验内存使用
  17. cat /proc/swaps 显示哪些swap被使用
  18. cat /proc/version 显示内核的版本
  19. cat /proc/net/dev 显示网络适配器及统计
  20. cat /proc/mounts 显示已加载的文件系统
  21. lspci -tv 罗列 PCI 设备
  22. lsusb -tv 显示 USB 设备
  23. date 显示系统日期
  24. cal 2007 显示2007年的日历表
  25. date 041217002007.00 设置日期和时间 – 月日时分年.秒
  26. clock -w 将时间修改保存到 BIOS
  27. shutdown -h now 关闭系统(1)
  28. init 0 关闭系统(2)
  29. telinit 0 关闭系统(3)
  30. shutdown -h hours:minutes & 按预定时间关闭系统
  31. shutdown -c 取消按预定时间关闭系统
  32. shutdown -r now 重启 (1)
  33. reboot 重启 (2)
  34. logout 注销
  35. free -m # 查看内存使用量和交换区使用量
  36. df -h # 查看各分区使用情况
  37. du -sh <目录名> # 查看指定目录的大小
  38. grep MemTotal /proc/meminfo # 查看内存总量
  39. grep MemFree /proc/meminfo # 查看空闲内存量
  40. uptime # 查看系统运行时间、用户数、负载
  41. cat /proc/loadavg # 查看系统负载
  42. mount | column -t # 查看挂接的分区状态
  43. fdisk -l # 查看所有分区
  44. swapon -s # 查看所有交换分区
  45. hdparm -i /dev/hda # 查看磁盘参数(仅适用于IDE设备)
  46. dmesg | grep IDE # 查看启动时IDE设备检测状况
  47. ifconfig # 查看所有网络接口的属性
  48. iptables -L # 查看防火墙设置
  49. route -n # 查看路由表
  50. netstat -lntp # 查看所有监听端口
  51. netstat -antp # 查看所有已经建立的连接
  52. netstat -s # 查看网络统计信息
  53. ps -ef # 查看所有进程
  54. top # 实时显示进程状态
  55. w # 查看活动用户
  56. id <用户名> # 查看指定用户信息
  57. last # 查看用户登录日志
  58. cut -d: -f1 /etc/passwd # 查看系统所有用户
  59. cut -d: -f1 /etc/group # 查看系统所有组
  60. crontab -l # 查看当前用户的计划任务
  61. chkconfig –list # 列出所有系统服务
  62. chkconfig –list | grep on # 列出所有启动的系统服务
  63. rpm -qa # 查看所有安装的软件包
  64. man <命令> #获得命令帮助
