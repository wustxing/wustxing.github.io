  • box-sizing

    在标准盒子模型中，元素的总宽＝content + padding + border + margin。

box-sizing属性就是用来重定义这个计算方式的，它有三个取值，分别是：content-box（默认）、border-box、padding-box

一般来说，假如我们需要有一个占宽200px、padding10px、border5px的div，经过计算，要这么定义样式。
